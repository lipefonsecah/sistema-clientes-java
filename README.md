# Sistema de Cadastro de Clientes em Java

Projeto desenvolvido para prática de Programação Orientada a Objetos (POO) utilizando Java durante o curso de Análise e Desenvolvimento de Sistemas.

## 📚 Descrição

Este sistema simula um cadastro simples de clientes através de um menu interativo no console.  
O usuário pode cadastrar clientes informando seus dados e visualizar as informações cadastradas.

O objetivo do projeto é praticar conceitos básicos de programação em Java, como:

- Classes e objetos
- Atributos
- Métodos
- Estrutura de repetição
- Entrada de dados com Scanner

## ⚙️ Funcionalidades

- Cadastrar cliente
- Informar nome e idade do cliente
- Exibir dados do cliente cadastrado
- Menu interativo no console

## 🛠 Tecnologias utilizadas

- Java
- Eclipse IDE

## ▶️ Como executar o projeto

1. Abrir o projeto no Eclipse
2. Executar a classe `Principal.java`
3. Utilizar o menu no console

Exemplo de funcionamento do sistema:


=== SISTEMA DE CLIENTES ===
1 - Cadastrar cliente
2 - Mostrar cliente
3 - Sair
