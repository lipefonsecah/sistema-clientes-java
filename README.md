# Sistema de Cadastro de Clientes em Java

Projeto desenvolvido para prática de Programação Orientada a Objetos em Java.

## Funcionalidades

- Cadastrar cliente
- Mostrar dados do cliente
- Menu interativo no console

## Tecnologias utilizadas

- Java
- Eclipse IDE

## Como executar

1. Abrir o projeto no Eclipse
2. Executar a classe Principal.java
3. Utilizar o menu no console
